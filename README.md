# MindLab-MindBits
Έργο της ομάδας MindLab-MindBits coders για τον 8ο ΠΑΝΕΛΛΗΝΙΟ ΔΙΑΓΩΝΙΣΜΟ ΑΝΟΙΧΤΩΝ ΤΕΧΝΟΛΟΓΙΩΝ ΣΤΗΝ ΕΚΠΑΙΔΕΥΣΗ

Παιχνίδι εκπαίδευσης μνήμης

Περιγραφή

Η ομάδα μας η οποία αποτελείται από μαθητές Γυμνασίου των Online εργαστηρίων του MindLab Education θα ασχοληθεί με την δημιουργία μίας εφαρμογής πρώτων βοηθειών. Πολλοί συνάνθρωποί μας παρουσιάζουν δυσκολίες μνήμης με αποτέλεσμα να δυσκολεύονται να διατηρήσουν ένα γεγονός στην μνήμη τους. Η κεντρική ιδέα της ομάδας μας είναι να δημιουργήσει μία εφαρμογή / παιχνίδι που θα εκπαιδεύει την μνήμη του ανθρώπου που παίζει. 

Στόχοι

•	Δημιουργία εφαρμογής μέσω Python και με χρήση βιβλιοθήκης p5 (Processing).
•	Γνωριμία σχετικά με την διαχείριση συμβάντων σε παιχνίδι.
•	Έλεγχος και αξιολόγηση του τελικής εφαρμογής / παιχνιδιού.
 
Σχεδιασμός-Υλοποίηση

Για την δημιουργία της του παιχνιδιού «Παιχνίδι εκπαίδευσης μνήμης» θα χρησιμοποιηθεί, από την ομάδα, η ιστοσελίδα www.trinket.io/turtle η Python και η βιβλιοθήκη p5. Αρχικά θα εμφανίζεται διαδοχικά μία σειρά χρωμάτων στην οθόνη του παίκτη. Στη συνέχεια θα πρέπει ο παίκτης να επιλέγει με τον κέρσορα τα χρώματα με την σειρά που τα είδε. Για κάθε πετυχημένη επιλογή θα προστίθεται και ένα ακόμη χρώμα για τον επόμενο γύρο. Στην περίπτωση αποτυχίας, τα χρώματα θα ξαναεμφανίζονται χωρίς προσθήκη νέου χρώματος στο τέλος της σειράς. Με αυτό τον τρόπο δεν υπάρχει ήτα παιχνιδιού αλλά επιπλέων εκπαίδευση της μνήμης.

Λογισμικά που θα χρησιμοποιηθούν:

1.	Python 3
2.	Βιβλιοθήκη p5
